## 项目所达到的愿景
> 打造一个可以用于任何spring boot项目的安全框架。
## 模块使用说明
- 本模块需要继承
```
    security-core
```
- 需要在主体项目中配置redis服务器，进行手机验证码登录或者图片验证码验证
- 使用本模块验证数据需要在header中添加```deviceId```
- 本模块主要实现了对于APP的认证部分 
## 项目所用框架
-     spring boot
-     spring security
-     spring social
-     spring session
-     redis缓存服务器
